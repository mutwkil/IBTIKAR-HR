Afag Loan Management

Manage Loan Request of Employees

Key Highlights

Loan Request

Employees can create loan requests.

Loan Request Approval/Refuse

Employee/Administrator can approve or reject the loan requests.

Pending Installments in Payslips

While creating Payslips, automatically lists the pending installments for the corresponding month.

Loan Amount Deduction From Salary

The loan amount will automatically deduct from the Salary.

Add Deduction Rule

Add a Salary deduction rule for Loan by selecting the Salary Rule Name as 'Loan'.

Loan Request

Go to Employees -> Loan & Advances -> Click on the 'Request for Loan' menu, where Employees can create a Loan Request.

Loan Request Form

Employees have the ability to create a Loan Request by providing details such as Loan Amount, Number Of Installments, and other relevant information.

Compute Installment

On clicking the "Compute Installment" button, will compute the Installments lines based on the Loan Amount and No Of Installments

Loan Request Approval

After submitting the Loan Request, the User having Employee/Administration right can approve the request by clicking the button "Approve" .

Employee Loan Requests

Able to view the Loan Requests related to a specific Employee by clicking the "Loans" smart button within the Employee Form view.

Pending Installments in Payslips

When creating the Payslip, all pending Installments for the month will be listed.

Installment Amount Deduction

After Confirming the Payslip, the Installment Amount will deducted from the Total Loan Amount
